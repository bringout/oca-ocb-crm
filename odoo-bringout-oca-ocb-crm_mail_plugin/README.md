# CRM Mail Plugin

Turn emails received in your mailbox into leads and log their content as internal notes.

## Installation

```bash
pip install odoo-bringout-oca-ocb-crm_mail_plugin
```

## Dependencies

This addon depends on:
- crm
- mail_plugin
